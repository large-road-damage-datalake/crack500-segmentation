# Crack500

## Overview

Crack500 is a task-specific segmentation package derived from the pavement-crack-detection project.

- Task: segmentation
- Images: 494
- Annotations: 5040
- Classes: 1 (crack)
- Annotation format: paired PNG masks

This local raw copy contains 494 valid image-mask pairs under the provided folder.
